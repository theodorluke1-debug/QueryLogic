QueryAI

Ask your database questions in plain German – no SQL needed.
Live: https://theodorluke1-debug.github.io/QueryAI

What it does

QueryAI lets you analyze your data by asking questions in natural language. No SQL knowledge required.

Import CSV or Excel files
Type your data directly into a spreadsheet
Connect your Supabase database
Ask questions in plain German or English
Get SQL generated automatically and executed
Results displayed as charts (bar, line, pie, KPI)
How to use

Open the app
Get a free Gemini API key at aistudio.google.com
Import your data or connect Supabase
Ask a question like "How many users signed up this week?"
Done
Tech stack

Vanilla JavaScript (no frameworks)
Gemini API (free tier)
sql.js (real SQLite in the browser)
Chart.js
SheetJS (Excel import)
